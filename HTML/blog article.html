note of html


一个简单的“我想建一个网站”的想法，重要的是你首先要弄清楚目标是什么，以及如何实现它。目标确定之后，你需要决定什么时候以及如何来实现它。将大的目标分成小的，较容易且可控制的目标，逐一实现它们，你就能达成那个在之前看起来是不可能的目标。

note of CSS


CSS 主要用于设置 HTML 页面中的文本内容、图片的外形以及版面的布局和外观显示样式。它让我们的网页更加丰富多彩，布局更加灵活自如。简单理解：CSS 可以美化 HTML , 让 HTML 更漂亮， 让页面布局更简单。

note of JavaScript

JavaScript一度被认为是一种玩具编程语言，它有缺陷，不被大多数后端开发人员所重视。有人认为，写JavaScript代码很简单，JavaScript只是为了在网页上添加一点交互和动画效果。 但这是完全错误的理解。JavaScript确实很容易上手，但其精髓却不为大多数人所熟知,而一名合格的开发人员应该精通JavaScript和其他编程语言。

style of uniforms


不少galgame的背景是在校园,因此,校服的设计至关重要,每个角色的校服都是相同样式,如何仍然突出人物特点,这是值得探讨的问题.不少废萌作品将校服画的比较华丽,色彩丰富,装饰颇多,个人认为不可取,校服为人设服务,不是校服本身越好看越好,而是越贴合人设越好,不少素雅的校服反而可以突出人物特点,成为点睛之笔.

theme of game


欲有游戏者,先有主题,如何选择一个主题,笔者有一定感悟:主题无非由以下子题结合而成,一是某种事物:如镜子/候鸟/向日葵,二是某种情感:亲情/爱情/复仇,三是一种哲学观点:如著名的"善恶相抵",其四则是一种妄想:比如"如果可以改变他人心灵世界会发生什么?"(女神异闻录5)/"如果入门律师上场打官司会怎么样?"(逆转裁判),主题是游戏的第一步,也是游戏的灵魂,需要重视.

color of hair

发色决定角色属性,是目前galgame中较为常见的一个现象,经历了多年的发展,由于各类发色对应的角色在某些属性上有强烈的共性存在,导致人们对发色产生了一定的固有印象,此中,白发和金发特为尤甚,再是其他发色,最后黑发作为我们一般人的发色,涵盖的属性较多,固有化程度低,但是仍有黑长直这种传统印象存在,笔者认为,发色影响属性对人设有利有弊,目前不妨顺势而行,但也要有打破固有印象的大胆创新.
